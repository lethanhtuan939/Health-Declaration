### Installation

#### FE (Angular)

Install packages

```
npm install
```

Run project

```
npm start
```

#### BE (Spring boot)

Create a database named healthdeclaration in MySQL

```
create database healthdeclaration;
```

Install dependencies

```
mvn install
```
